# Estudos de Python: Ciência de Dados & Big Data Analytics 🚀

Este repositório contém meus scripts de estudo e exercícios práticos desenvolvidos durante a pós-graduação em **Ciência de Dados e Big Data Analytics**. O objetivo aqui é documentar minha evolução na linguagem Python, desde a lógica fundamental até a manipulação de dados.

## 📂 Estrutura do Repositório

Os arquivos estão organizados por temas de estudo, cobrindo fundamentos e aplicações iniciais:

### 🐍 Fundamentos de Python
* **TipagemIntBoolFloat.py**: Exercícios sobre tipos de dados primitivos.
* **Binding.py**: Estudos sobre como o Python lida com referências de objetos e atribuições.
* **Hello.py**: O clássico ponto de partida para testar o ambiente.
* **Testes.py**: Script geral para experimentação de novos comandos.

### 🔢 Lógica e Matemática
* **MediaNotas.py**: Cálculo de médias simples.
* **MediasInput.py**: Script interativo para cálculo de médias com entrada de dados do usuário.
* **CaixaSup.py**: Possível lógica para sistemas de caixa ou operações matemáticas específicas.

### 📊 Manipulação e Visualização de Dados
* **LocPandas.py**: Práticas com a biblioteca Pandas, focando no método `.loc` para filtragem de dados.
* **Graficos.py**: Geração de visualizações e plots estatísticos.
* **CMSatisfacao.py**: Script focado em análise de métricas de satisfação (Customer Metrics).
* **PersonaSocial.py**: Análise de dados voltada para perfis sociais ou personas.

### 🌌 Projetos Temáticos
* **SisSolar.py** & **dados_sistema_solar**: Script e base de dados para análise de informações astronômicas do nosso sistema solar.

### 🛠️ Arquivo Principal
* **main.py**: Ponto de entrada para execução de rotinas principais ou integração dos módulos.

---

## 🛠️ Tecnologias Utilizadas
* **Python 3.x**
* **Pandas**: Para manipulação de tabelas e dados estruturados.
* **Matplotlib/Seaborn**: (Provável) para a geração de gráficos.
